Interactions are messages that will be displayed to the user. Combine them with User data actions, Calculations, and Filters to create complex Collections.

To use dynamically generated content in your messages, use Insert special text when viewing the collection.

{{/}} {{# name === 'user-data'}}

User data actions are used with an Interaction to be able to use a user field later, or to categorise a user.

{{/}} {{# name === 'placeholders'}}

Placeholders are blocks of dynamically generated content that you can use inside an Interaction block.

Use them by putting their keyword inside square brackets like this: [keyword]
